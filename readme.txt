This project, Car price, is a Django Data science based application.
The aim is to create a web app which can predict the price of a car based on certain mix of information
about the car. 

Steps:
Modelling:
Obtain training car dataset of over 4000 data point. Each data point has the following infomation:
brand, engine type, body, price, registration, mileage,model and engineV
Preprocess the data ready for modelling
Train the data and obtain regression model

Frontend:
create an html page in Django with form where user can send in input
create and serve the CSS files or static files

Backend:
Backend which accept the user input
deploy the trained model for serving in Django
process the user input with the model and return predicted price to the user
